# Security Policy

## Supported versions

Tensotron is pre-1.0; only the latest published `0.1.x-alpha` release receives fixes.

| Version     | Supported          |
|-------------|--------------------|
| 0.1.x-alpha | yes (latest only)  |
| < 0.1.0     | no                 |

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue.

- Preferred: open a private security advisory on GitHub (**Security → Report a vulnerability**), or
- Email **Mattias.Fagerlund@carretera.se** with details and a reproduction.

You'll get an acknowledgement as soon as possible. As a spare-time OSS project, please allow
reasonable time for a fix before any public disclosure.
